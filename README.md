# صلة (Sıla)

**صلة** هو تطبيق إسلامي عصري، فاخر، وتفاعلي مصمم خصيصاً للشباب (الناطقين بالعربية والتركية). يهدف التطبيق إلى تسهيل العبادات اليومية (القرآن، الأذكار، الصلاة) مع التركيز على سهولة الاستخدام، القيمة التعليمية، والمشاركة "النشطة" لمنع الملل.

## 🌟 الفلسفة
1. **الصلة بالله**: حبلك الموصول بالله عبر الذكر والقرآن.
2. **الصلة بالأحباب**: دمج مفهوم "صلة الرحم" والوفاء للأموات والأصدقاء من خلال إهداء الثواب والدعاء.

## 🚀 الميزات الرئيسية

### 1. قارئ القرآن النشط 📖
- تجربة قراءة ديناميكية مع تفسير فوري لكل صفحة.
- ميزة "ساعدني أقرأ" للاستماع لنطق الآيات والكلمات عند اللمس.
- إمكانية تسجيل تدبرات صوتية وملاحظات شخصية.

### 2. الأذكار الذكية والورد 📿
- اقتراح أذكار بناءً على الحالة الشعورية للمستخدم (قلق، سعيد، متعب).
- عداد تفاعلي مع تغذية بصرية ممتعة.
- شرح "لماذا؟" لكل ذكر لتعزيز التربية القيمية.

### 3. نظام الوفاء والتلعيب 🤝
- **قائمة الوفاء**: تذكير بالدعاء وإهداء الثواب للأحباب المتوفين.
- تتبع "نقاط الجنة" والمحافظة على "Streaks" (مثل حارس الفجر).
- ختمات قرآنية جماعية مهداة.

### 4. تصميم عصري فاخر 🎨
- واجهة مستخدم تعتمد على الزجاجية (Glassmorphism) والتدرجات الناعمة.
- خلفيات ديناميكية تتغير مع أوقات الصلاة.
- خطوط عالية الجودة للقراءة المريحة.

### 5. التسميع الذكي (Tasmi) 🎙️
- تصحيح فوري للتلاوة مع تمييز الكلمات الصحيحة، القريبة، والخاطئة.
- استقرار أعلى لخدمة الاستماع الصوتي (STT) مع إعادة تشغيل تلقائية محسّنة عند الصمت أو الانقطاع.
- تنسيق بين الاستماع (STT) والنطق (TTS) لمنع احتساب صوت التصحيح كخطأ على المستخدم.
- فقاعة تصحيح أوضح وتجربة متابعة تلقائية أكثر سلاسة.
- شاشة نتائج متقدمة: نسبة دقة، إحصائيات مفصلة، وقائمة كلمات تحتاج مراجعة.
- اقتراح إهداء الثواب في نهاية جلسة التسميع.

### 6. تفضيلات شخصية للتسميع ⚙️
- اختيار سلوك التطبيق عند الخطأ:
  - ينطق ويكمل تلقائياً.
  - ينتظر المستخدم.
  - يكمل بصمت.
- تحديد عدد المحاولات قبل تسجيل الخطأ (مرة / مرتان / ثلاث مرات).
- التحكم في تفعيل صوت التصحيح (TTS).
- اختيار مستوى الصرامة في المطابقة (متساهل / متوسط / صارم).
- حفظ التفضيلات محلياً عبر SharedPreferences مع إمكانية تعديلها لاحقاً.

## 🛠️ التقنيات المستخدمة
- **Framework**: Flutter
- **Language**: Dart
- **State Management**: Riverpod
- **Database**: Isar (Offline-first)
- **Localization**: Easy Localization (Arabic & Turkish)
