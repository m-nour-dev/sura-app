# Adhan Audio Files - Download Instructions

## Required Audio Files

Download the following Adhan MP3 files and place them in `assets/audio/`:

### 1. Adhan Mecca (adhan_mecca.mp3)
- **Source**: https://archive.org/details/adhan-notifications
- **Duration**: ~3-4 minutes
- **Quality**: 128kbps or higher

### 2. Adhan Medina (adhan_medina.mp3)
- **Source**: https://archive.org/details/adhan-notifications  
- **Duration**: ~3-4 minutes
- **Quality**: 128kbps or higher

### 3. Adhan Egypt (adhan_egypt.mp3)
- **Source**: https://islamcan.com/azan-adhan-download.shtml
- **Duration**: ~3-4 minutes
- **Quality**: 128kbps or higher

### 4. Adhan Mishary (adhan_mishary.mp3)
- **Reciter**: Mishary Rashid Al-Afasy
- **Source**: https://archive.org/details/adhan-notifications
- **Duration**: ~3-4 minutes
- **Quality**: 128kbps or higher

### 5. Adhan Turkey (adhan_turkey.mp3)
- **Source**: https://islamcan.com/azan-adhan-download.shtml
- **Duration**: ~3-4 minutes  
- **Quality**: 128kbps or higher

## Download Steps

1. Visit the sources above
2. Download MP3 files
3. Rename them to match the names above
4. Place all files in: `C:\Users\Hp\OneDrive\Desktop\2026 plan\Sila\sila_app\assets\audio\`
5. Run `flutter pub get` to refresh assets
